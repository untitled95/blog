Github Pages

github pages 还是很简单的，创建一个以 名字.github.io 结尾的 repository，名字什么的自己随便叫，但是要是英文的

这个 repo 里面的东西就是会显示在博客上的啦，如果你不想绑定自己的域名，那么 名字.github.io 就会是你自己的网站啦。 不需要复杂功能的话只需要在这个repo里建一个 README.md 文件然后修改这个文件，就可以在自己的网站上修改博客了。

如果想链接自己的域名怎么办呢

  1. 在repo里加一个CNAME文件，全大写
  2. 在CNAME里面写上你的域名，例如我就是lyiming.io
  3. 打开自己的域名管理，我用的是google domain
  4. 在domain的 DNS 里面加上新的DNS设置
