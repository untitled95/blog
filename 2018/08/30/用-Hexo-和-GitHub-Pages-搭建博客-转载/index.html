用 Hexo 和 GitHub Pages 搭建blog
Post：2018-08-30 12:55:00 Tags：/ blog / Visit：

我参考了这个文章 [用 Hexo 和 GitHub Pages 搭建博客]
https://ryanluoxu.github.io/2017/11/24/用-Hexo-和-GitHub-Pages-搭建博客/
搭建了我自己的blog

前期准备

1.github pages

其实这个非常的简单，主要是在链接自己的域名的时候遇到一些问题
注意 CNAME 和 A 就可以了

2.Hexo 框架

我发现这个框架还是很好用的，所以大概以后博客类型的网页我都会用这个啦
其实就是帮你生成了html，css之类的东西，比自己手写还是快多了，想当初自己手写html简直想哭

3.Github Pages

github pages 还是很简单的，创建一个以 名字.github.io 结尾的 repository，名字什么的自己随便叫，但是要是英文的

这个 repo 里面的东西就是会显示在博客上的啦，如果你不想绑定自己的域名，那么 名字.github.io 就会是你自己的网站啦。
不需要复杂功能的话只需要在这个repo里建一个 README.md 文件然后修改这个文件，就可以在自己的网站上修改博客了。

如果想链接自己的域名怎么办呢

4.链接域名相关

1.在repo里加一个CNAME文件，全大写

在CNAME里面写上你的域名，例如我就是lyiming.io

2.打开自己的域名管理，我用的是google domain

在domain的 DNS 里面加上新的DNS设置
